💡 Como organizar e subir no GitHub Pages?

  1. Crie um repositório no GitHub para guardar seus projetos (ex: meus-treinamentos).
  2. Suba este arquivo com o nome de index.html na pasta raiz do repositório.
  3. Suba os arquivos de cada treinamento na mesma pasta (ex: salve o treinamento Hisense como treinamento-hisense.html no mesmo local).
  4. No GitHub, vá em Settings → Pages, selecione a branch main (ou master) e a pasta / (root), e clique em Save.
  5. O GitHub gerará um link público (ex: https://seu-usuario.github.io/meus-treinamentos/). Esse link abrirá este Portal, e a partir dele todos os outros links funcionarão perfeitamente!
